# Shared Components

Reusable SwiftUI view components that serve multiple features and are not owned by a single feature area.

## Current Components

- `BrandMarkView` — renders the lantern brand mark at configurable sizes
- `ConnectivityBadge` — displays connectivity state indicator

## Adding Components

Promote view code here only when at least two app areas genuinely need the same component. Keep feature-specific views in their respective feature folders.
