# Lantern Brand Source Notes

- The app icon master is derived from `scripts/brand/lantern_asset_source.html`.
- The core palette is aligned to the existing asset catalog colors in `OSA/Resources/Assets.xcassets/`.
- The auxiliary monogram is a simplified lantern mark created for favicon, compact-nav, and documentation use where the full app icon is too dense.
- The wordmark uses the shipped Lantern naming already present in the repository README and `AppBrand.swift`.
