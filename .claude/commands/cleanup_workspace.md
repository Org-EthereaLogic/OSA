# cleanup_workspace

allowed-tools: Bash, Read, Glob, Grep

Remove build artifacts and temporary files from the OSA workspace. Default mode is **dry-run** (preview only).

## Variables

mode: $ARGUMENTS

## Cleanup Targets

| Target | Description | Regenerated By |
|--------|-------------|----------------|
| `DerivedData/` (project) | Xcode build cache | `xcodebuild build` |
| `.build/` | SPM build directory | `swift build` |
| `.swiftpm/` | SPM workspace state | `swift package resolve` |
| `*.xcresult` | Test result bundles | `swift test` |
| `.DS_Store` (recursive) | macOS Finder metadata | macOS Finder |
| `*.log` (root level) | Log files | Various processes |
| `logs/hooks/` | Claude hook logs | Hook scripts |

## Protected Paths (NEVER remove)

`.git/`, `Sources/`, `Tests/`, `docs/`, `.claude/`, `.github/`, `Package.swift`, `README.md`, `*.xcodeproj`, `*.xcworkspace`

## Workflow

### Dry-Run (default — no `--execute` flag)

1. Scan for cleanup targets
2. Calculate sizes with `du -sh`
3. Report what WOULD be removed with sizes
4. Show total recoverable space
5. Show command to execute: `/cleanup_workspace --execute`

### Execute (`--execute` flag provided)

1. Verify targets exist before removal
2. Remove each target
3. Report each removal with size
4. Show total space recovered

## Safety Rules

1. Check target exists before removal
2. Never remove protected paths
3. Never remove git-tracked files
4. Stop on any error
5. Dry-run is always the default
